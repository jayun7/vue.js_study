# vue.js_study

## Setup & Build

```
$ npm install
$ gulp build
```

## Only Build SASS 

```
$ gulp build:sass
```

## Build with Uncompression & Unminify

```
$ DEBUG=Y gulp build
```

## Build with Compression & Uglify 

```
$ DEBUG=N gulp build
```
OR

```
$ gulp build
```

## Running HTML File

```
$ PUBLISH=Y gulp build
```

## Running Local Web Server

```
$ npm start
```
